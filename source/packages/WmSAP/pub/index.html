webMethods Adapter for SAP Package
The WmSAP Package is used to host webMethods Adapter for SAP.

1 Documentation

1.1 Adapter Documentation

The following documentation is available for the adapter:

  • webMethods Adapter 10.1 for SAP Release Notes
  • webMethods Adapter 10.1 for SAP readme
  • webMethods Adapter 10.1 for SAP Installation and User's Guide

Where can I find the documentation?

Documentation is available from the Software AG Documentation Web site and from the Installer:

  • Visit the webMethods area of the Software AG Documentation Web site for the latest updates to the Adapter for SAP documentation. The Web site also contains documentation for other products, documentation for previous versions of our products, and documentation in other languages.
  • As of version 9.0 of the webMethods product suite, product documentation is also available for download from the webMethods Installer. From the Installer, select "Documentation" to download and install the entire webMethods product library, or navigate within the Documentation folder to select the product libraries to install. By default, the Installer installs the documentation files to a single directory called "_documentation," located under your webMethods product installation directory.

1.3 Java Documentation

The Adapter 10.1 for SAP uses the SAP Java IDoc Class Library to handle IDocs. However, creating, sending, and receiving IDocs is middleware dependent. Therefore, the Adapter for SAP provides a Java API that implements the interfaces defined by the SAP Java IDoc Class Library. The provided JavaDoc explains how to use that API.

  • webMethods Adapter for SAP IDoc Java API

3 Migration

The webMethods Adapter for SAP Installation Guide provides step by step instructions to migrate your existing SAP Adapter 4.6 based application packages to use Adapter 10.1 for SAP. This is an easy to accomplish approach for an environment where you have defined a small set of SAP Adapter 4.6 constructs (for example, inbound maps) only.

To automatically migrate your SAP Adapter 4.6 setup for use with Adapter 10.1 for SAP, the following HTML form is provided:
